# Student Management System

This is a simple Java-based Student Management System project developed to manage student records efficiently.

## 📁 Project Files
- Student.java → Model class (stores student details like ID, name, marks, etc.)
- StudentManager.java → Business logic (add, view, search, update student records)
- main.java → Main class (runs the application)

## ✨ Features
- Add new student
- View all students
- Search student by ID
- Update student details

## ⚙️ Technologies Used
- Java
- OOP Concepts
- File Handling

## ▶️ How to Run
1. Compile all files:
   javac *.java

2. Run the main file:
   java main
